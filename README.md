# Axiom OS — Research Data & Analysis

Supplementary datasets and reproducibility notebook for the 
submitted manuscript on per-read BLAKE3 provenance enforcement.

## Contents

- `axiom_data_analysis.ipynb` — automated analysis pipeline. 
  Generates all motivation figures and benchmark statistics 
  reported in the paper. No hardcoded values.
- `mobile_prices_2023.csv` — Flipkart India smartphone listings 
  (N=1,836, June 2023). Source: Kaggle/Howisusmanali.
- `android_version-IN-monthly-202502-202602.csv` — Android 
  version market share India. Source: Statcounter GlobalStats.
- `Cyber_crimes_against_children_NCRB_data_TABLE_9A_11__IPC___SLL__for_years_2017_to_2021_with_child_population.xlsx` 
  — NCRB cybercrime against children 2017-2021. 
  Source: Mendeley Data (CC BY 4.0).

## Reproducing the Analysis

Upload all three dataset files to Google Colab alongside 
the notebook and run all cells. All figures and statistics 
generate automatically.
